## ‚öôÔ∏è Installation & Setup

1.  **Clone the repository**
    ```bash
    git clone https://github.com/DIVYANSH-675/DIVYANSH-675.github.io.git
    cd DIVYANSH-675.github.io
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

    ```bash
    npm run dev
    ```

4.  **Build for production**
    ```bash
    npm run build
    ```
